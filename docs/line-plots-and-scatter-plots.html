Common Parameters of 'plot' Function
color: Sets the color of the line.
linestyle: Sets the line style, e.g., solid, dashed, etc.
linewidth: Sets the thickness of a line.
marker: Chooses a marker for data points, e.g., circle, triangle, etc.
markersize: Sets the size of the chosen marker.
label: Names the line, which will come in legend.
For customizing line, required parameters need to be passed as arguments toÂ plotÂ function.
AÂ green dashed line, having widthÂ 3Â can be generated by using the following expression.

Scatter plotÂ is very similar toÂ Line Plot.
Scatter PlotÂ is used for showing how one variable is related with another.
Scatter PlotÂ consists of data points. If the spread of data points is linear, then two variables are highly correlated.
scatterÂ function is used for drawing scatter plots.

Common Parameters of 'scatter'
c: Sets color of markers.
s: Sets size of markers.
marker: Selects a marker. e.g: circle, triangle, etc
edgecolor: Sets the color of lines on edges of markers.
ParametersÂ cÂ andÂ sÂ can take a list of values.
If the number of values is less than the number of data points considered, then the list is repeated.
The below example plotsÂ greenÂ coloredÂ circlesÂ of sizeÂ 60, withÂ black edges.
