Example
def multiple_of(x): def multiple(y): return x*y return multiple c1 = multiple_of(5) # 'c1' is a closure c2 = multiple_of(6) # 'c2' is a closure print(c1(4)) print(c2(4))
Python

The first closure function,â€¯c1â€¯binds the valueâ€¯5â€¯to argumentâ€¯xâ€¯and when called with an argumentâ€¯4, it executes the body ofâ€¯multipleâ€¯function and returns the product ofâ€¯5â€¯andâ€¯4.
