Diferença para a Regressão Linear
Regressão Linear busca minimizar os resíduos (soma dos mínimos quadrados). Ridge busca minimizar os resíduos + Lambda * Inclinação^2
Lambda é uma penalidade que leva para o lado do Bias
Regularização: Lasso (L1)
Quando usar?
Quando há indicação de que se pode eliminar features sem importância
Anotações
Diferença para a Ridge está na fórmula:
Regressão Linear busca minimizar os resíduos (soma dos mínimos quadrados). Ridge busca minimizar os resíduos + Lambda * |Inclinação|
Além disso, Lasso pode efetivamente levar a inclinação até o 0 (Ridge leva até muito perto de 0)
Lambda é uma penalidade que leva para o lado do Bias
