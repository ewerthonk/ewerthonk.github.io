Histogram
HistogramÂ is used to visualize the spread of data of a distribution.
histÂ function is used to plot a histogram.
Syntax
hist(x) # 'x' : Data values of a single variable.

Histogram Using 'hist'
The below example simulatesÂ 1000Â percentage values from a normal distribution with meanÂ 60Â and standard deviationÂ 10.
Then the histogram of percentage values is plotted.
import numpy as np np.random.seed(100) x = 60 + 10*np.random.randn(1000) fig = plt.figure(figsize=(8,6)) ax = fig.add_subplot(111) ax.set(title="Distribution of Student's Percentage", ylabel='Count', xlabel='Percentage') ax.hist(x) plt.show()

Common Parameters of 'hist'
color: Sets the color of bars.
bins: Sets the number of bins to be used.
normed: Sets toÂ TrueÂ where bins display fraction and not the count.
Setting Parameters of 'hist'
You can also create more bins and show bin count as a fraction (as specified in the following example).
import numpy as np np.random.seed(100) x = 60 + 10*np.random.randn(1000) fig = plt.figure(figsize=(8,6)) ax = fig.add_subplot(111) ax.set(title="Distribution of Student's Percentage", ylabel='Proportion', xlabel='Percentage') ax.hist(x, color='blue', bins=30, density=True) plt.show()

Box Plots
Box plotsÂ are also used to visualize the spread of data.
Box plotsÂ are used to compare distributions.
Box plotsÂ can also be used to detect outliers.
Syntax
boxplot(x) # 'x' : list of values or list of list of values.
Copy
Python
Boxplot Using 'boxplot'
The above image displays box plot of Percentages obtained byÂ 1000Â Students of a class.
import numpy as np np.random.seed(100) x = 50 + 10*np.random.randn(1000) fig = plt.figure(figsize=(8,6)) ax = fig.add_subplot(111) ax.set(title="Box plot of Student's Percentage", xlabel='Class', ylabel='Percentage') ax.boxplot(x) plt.show()
Copy
Python
Common Parameters of 'boxplot'
labels: Sets the labels for box plots.
notch: Sets toÂ TrueÂ if notches need to be created around the median.
bootstrap: Number set to indicate that notches around the median are bootstrapped.
vert: Sets toÂ FalseÂ for plotting Box plots horizontally.
Setting Parameters of 'boxplot'
Box plot of Student Percentages can be redrawn by settingÂ notch,Â bootstrapÂ andÂ labelsÂ using the below-shown expression.
ax.boxplot(x, labels=['A'], notch=True, bootstrap=10000)

Plotting Multiple Boxplots
List of data values can be passed as an argument for plotting multiple box plots as shown in below code snippet.
import numpy as np np.random.seed(100) x = 50 + 10*np.random.randn(1000) y = 70 + 25*np.random.randn(1000) z = 30 + 5*np.random.randn(1000) fig = plt.figure(figsize=(8,6)) ax = fig.add_subplot(111) ax.set(title="Box plot of Student's Percentage", xlabel='Class', ylabel='Percentage') ax.boxplot([x, y, z], labels=['A', 'B', 'C'], notch=True, bootstrap=10000) plt.show()
